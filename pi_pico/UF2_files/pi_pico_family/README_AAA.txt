FLASH NUKE
The flash nuke file recovers a bricked Pi Pico by erasing the flash.
It works for both RP2040  and  RP2350 chips.
   https://datasheets.raspberrypi.com/soft/flash_nuke.uf2
From the official page:
  https://www.raspberrypi.com/documentation/microcontrollers/pico-series.html#resetting-flash-memory
