Adafruit STEMMA Soil Sensor

Google search:
Unlike the SparkFun sensor, the Adafruit board is engineered around the ATSAMD10 microcontroller. This chip contains a robust, dedicated hardware I2C engine. It doesn't rely on flaky software interrupts to translate bits, it won't crash if the bus has a voltage glitch, and it handles clock signals flawlessly.Bonus features: Because it is a true micro-controller handling the lines, it natively supports CircuitPython/MicroPython, won't oxidize over time, and even provides free ambient temperature tracking on the side.

Not long wire friendly:  6-8 inches MAX.
